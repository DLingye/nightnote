# NightNote - 轻量级文本编辑器

NightNote是一个简单易用的文本编辑器，支持多缓冲区管理和基本编辑功能。

## 功能特点

- 多缓冲区管理
- 基本的文本编辑功能
- 查找和替换
- 支持撤销/重做
- 状态栏显示编辑信息
- 支持深色/浅色主题

## 安装

1. 确保已安装Python 3.6+
2. 安装依赖：
   ```bash
   pip install PyQt6
   ```

## 使用方法

### 基本操作
- 新建文件: `Ctrl+N`
- 打开文件: `Ctrl+O`
- 保存文件: `Ctrl+S`
- 另存为: `Ctrl+Shift+S`

### 编辑功能
- 撤销: `Ctrl+Z`
- 重做: `Ctrl+Y`
- 查找: `Ctrl+F`
- 替换: `Ctrl+H`

### 缓冲区管理
- 新建缓冲区: `Ctrl+B`
- 切换缓冲区: `Ctrl+Tab`
- 列出缓冲区: 菜单栏->缓冲区->列出缓冲区
